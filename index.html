開發環境

擴增實境：以 Unity 設計擴增實境app 並安裝於 Android 手機或平板。

(若本身有 Apple 的開發者帳號和 OSX 的系統也可以 iPhone / iPad 實作)

  • Android 手機或平板
  • Unity - 傳送門 (請裝5.3.0以下的版本,配合Vuforia 5.0.1)
  • JAVA SDK - 傳送門
  • Android SDK - 傳送門
  • Vuforia SDK - 傳送門
  • NGUI 從Unity Assets store ($95) 或者 試用一下，!Wj9x4rTiYc_JxvG7uU1AM_7_tx6NjLL80rTky2TUXYc

體感互動：以 Unity 設計互動遊戲，硬體使用 Kinect 或 Kinect V2 連接 PC。

使用 Kinect V2

  • Kinect V2 - 購買, 轉接線購買
  • Window 8 以上 需 USB3.0
  • Unity - 傳送門
  • Kinect SDK 2 - 傳送門

使用 Kinect

  • win7+
  • kinect for windows
  • kinect for windows SDK 1.8 - 傳送門
  • unity kinect plugin - 傳送門

手勢互動：以 Unity 設計互動遊戲，硬體使用 Leap Motion 連接 PC。

(需與學校討論使用 Leap Motion 或 以 Kinect)

  • Windows 7 64-bit or higher
  • Leap Motion - 購買
  • Unity - 傳送門
  • Leap Motion SDK 傳送門

  • Leap Motion Unity Assets - 傳送門 或 unity Leap Motion plugin 3.0 - 傳送門
